important HTML tags

This page is designed as a quick refrence for some of the common tags coverd in Learn Enough HTML to Be Dangerous . In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables bellow don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1-h6 headings include a heading (levels1-6)
p paragraph includes a paragraph of text
tabel tabel includes a table
tr table row includes a row of table
th table header make a tabe header
td table data includes a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
a anchor add a hyper link <a href="www.Google.com">Google</a> Google
strong stong makes the text bold <strong>Hello</strong> Hello
img Image Displays an image <img src="images/image-name.jpg" alt="image description"/> An image displays
code code makes the font look like code <code>word</code> word
